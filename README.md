# Brace Specialist Website

Professional orthodontist website for Dr T.A. Mvundla in Potchefstroom, South Africa. Specializing in braces, early treatment, and smile correction for children, teens, and adults.

## Features
- Expert orthodontic care information
- Free consultation booking system
- Responsive design for all devices
- Service descriptions and testimonials
- FAQ section addressing common questions
- Contact information and appointment form

## Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/nizaamj/brace-specialist-website.git
   ```
2. Open `index.html` in your web browser
